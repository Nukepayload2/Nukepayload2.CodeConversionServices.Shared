# Nukepayload2.CodeConversionServices.Shared
帮助VB写的WPF项目移植为UWP，并且帮助其它语言代码转换为VB代码<br/>
本项目需要Nuget包System.Text.Encoding.CodePageshe和一些项目级别导入, 否则无法正常编译。<br/>
如果用C#的网页想用这个代码库，新建一个vb的uwp类库，然后添加共享项目引用再编译就行了。<br/>
